# 📊 Sales Dashboard

Dashboard desenvolvido no Power BI utilizando os dados do projeto Sales Analytics.

## Objetivos

- Acompanhar faturamento
- Visualizar vendas por categoria
- Analisar vendas por estado
- Identificar produtos mais vendidos
- Monitorar KPIs comerciais

## KPIs

- 💰 Receita Total
- 📦 Total de Pedidos
- 👥 Clientes Atendidos
- 🎯 Ticket Médio

## Visualizações

- Receita Mensal
- Receita por Estado
- Top 10 Produtos
- Receita por Categoria
- Evolução das Vendas

## Tecnologias

- Power BI
- DAX
- Power Query
